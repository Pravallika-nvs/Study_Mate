# Study-Mate

AI-powered PDF learning assistant built using:

- Streamlit
- Gemini 2.5 Flash
- FAISS
- HuggingFace Embeddings
- PyPDF2

## Features

- Chat with PDFs
- Conversational Memory
- Semantic Search
- Multi-PDF Support

## Setup

pip install -r requirements.txt

Create a .env file:

GOOGLE_API_KEY=your_api_key

Run:

streamlit run app.py
